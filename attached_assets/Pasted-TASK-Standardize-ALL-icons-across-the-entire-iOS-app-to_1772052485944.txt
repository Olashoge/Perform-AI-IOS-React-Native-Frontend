TASK: Standardize ALL icons across the entire iOS app to a single OUTLINE icon system with uniform sizing/weight and zero filled icons.

CONTEXT
- This is the Perform AI (React Native + Expo) app.
- Current problem: icons are mixed (filled vs outline, multiple families), inconsistent sizes/weights.
- Goal: every icon in the app must be outline style and consistent.

NON-NEGOTIABLE REQUIREMENTS
1) Use ONE icon library across the entire app (no mixing).
2) All icons must be outline style (no filled icons anywhere).
3) Enforce uniformity via a single wrapper component:
   - Standard sizes only: 16 / 20 / 24 / 28
   - Centralized color tokens only (no random hex values in components)
4) Replace all direct icon usage across the codebase with the wrapper component.
5) Ensure tab bar icons, list-row icons, card icons, empty-state icons all match the same system.

ICON LIBRARY CHOICE
- Use lucide-react-native as the single icon system (outline-first, consistent geometry).
- If lucide-react-native cannot be installed for any reason, fallback to Feather from @expo/vector-icons, but still enforce a wrapper and no mixing.

A) Install and configure

- Wrapper should NOT expose strokeWidth changes per-call; set strokeWidth centrally (ex: 2).
